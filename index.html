Create new wallet

If you want to buy or sell someting you need to register. If you want to register you'll need a wallet, so that you can pay a small registration fee. We can create a new wallet for you. It'll be password encrypted so please provide a password. Don't forget to put some eth in the wallet before registering.

Please save your wallet information for later recovery.

Import wallet

You can also import your own wallet. Your private key will be stored only in the current browser and won't be sent anywhere else. Please provide a password so that your wallet is safely saved in the browser.

Register

After adding a wallet you can finally register.

Add new item for sale

Once you're registered you can start adding items for sale.
